description: 为新模块生成 TypeScript 接口类型定义模板
---

## 步骤

1. **确认信息**
   确认：模块名称、核心字段、API 接口列表、是否有状态流转

2. **生成类型定义**
   在项目类型目录中创建 `{module}.ts`
   - 包含实体类型、请求/响应类型
   - 添加 JSDoc 注释

3. **生成状态机（如需要）**
   如有状态流转，生成 `{module}-status.ts`：
   - 状态枚举
   - 流转规则（当前状态 → 允许的下一状态列表）
   - 各状态允许的操作

4. **更新统一导出**
   更新类型目录的 `index.ts` 统一导出

// turbo
5. **类型检查**
   运行项目的 typecheck 命令确认无错误
