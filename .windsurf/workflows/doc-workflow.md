# 文档工作流

> 根据当前文档模式（单人/团队）执行对应的文档工作流。

## 步骤

// turbo
1. **检测当前文档模式**
   读取 `docs/.doc-mode` 文件：
   - `solo` → 单人模式
   - `team` → 团队模式
   - 不存在 → 默认单人模式

2. **单人模式工作流**
   - 新页面：建议创建 Page Spec（非强制）
   - 新功能：建议创建设计文档（非强制）
   - 重构：建议更新架构文档

3. **团队模式工作流**
   - 新页面：**必须**先创建 `docs/pages/*.spec.md`
   - 新功能：**必须**先更新相关文档
   - 重构：**必须**先更新架构文档
   - 提交前自动运行 `pnpm doc-check`

4. **文档完整性检查**
   // turbo
   运行 `node scripts/doc-sync-check.js` 检查文档与代码的同步状态

5. **输出结果**
   列出文档状态和需要补充的文档清单。
