# AI 工程化健壮性检查

> 运行 7 维度自动检查，输出评分和改进建议。

## 步骤

// turbo
1. **运行健壮性检查脚本**
   ```bash
   node scripts/ai-health-check.cjs
   ```

2. **分析结果**
   - 查看每个维度的评分和待改进项
   - 重点关注 ❌ 和 ⚠️ 标记的维度

3. **给出改进建议**
   按优先级列出具体改进行动：
   - P0：立即修复（评分 < 6 的维度）
   - P1：短期改进（评分 6-8 的维度）
   - P2：持续优化（评分 ≥ 8 的维度）

4. **可选：JSON 格式输出**
   ```bash
   node scripts/ai-health-check.cjs --json
   ```
   适用于 CI 集成或自动化处理。
