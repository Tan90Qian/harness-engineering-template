## 通用规则

- 遵循 TypeScript strict 模式
- 提交代码遵循 Conventional Commits 规范

## 代码质量

- 所有共享类型必须从统一入口导出
- 优先组合而非继承
- 禁止使用 `any`，除非附带注释说明原因
- 所有公开函数必须有 JSDoc 注释
- 错误处理必须显式，禁止静默吞掉异常

- 工具函数必须有单元测试
- API 接口必须有集成测试
- 核心用户流程应有 E2E 测试
